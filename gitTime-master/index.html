Text Align

A propriedade text-align é um tipo de conteúdo inline, como texto, é alinhado no elemento pai em bloco.
text-align não controla o alinhamento de elementos em bloco, apenas seu conteúdo inline.

Exemplos:

text-align: left;

text-align: center;

text-aling: right;

Text Shadow

A propriedade text-shadow acrescenta sombras ao texto.
Ela aceita uma lista de sombras separadas por vírgula que serão aplicadas ao texto e ao text-decoration do elemento.

Exemplo:

/* deslocamento-x | deslocamento-y | raio-de-desfoque | cor */
text-shadow: 5px 5px 5px black;

Keyframes

As @keyframes especifica uma animação de código.
A animação é criada mudando gradualmente de um conjunto de estilos CSS para outro.

Especifique quando a mudança de estilo ocorrerá em porcentagem ou com as palavras-chave "from" e "to", que são iguais a 0% e 100%. 0% é o início da animação, 100% é quando a animação é concluída.

Exemplo:

@keyframes move {
from {left: 0px;}
to {left:610px; background:#7159C1;} }

FONTES:

A estilização das fontes são usadas para determinar a forma como que o texto se apresentara na tela, com estilo, tamanho e cor.


Font Family

=>A propriedade font-family serve para definir a fonte que será usada no texto geral, na qual o navedador escolherá a que melhor irá se comportar entre o conjunto aplicado nos elementos selecionados. Um exemplo de fonte utilizadas é a: Arial, Helvetica, sans-serif.

EXEMPLO:

font-family: Arial, Helvetica, sans-serif


Font Size

=>A propriedade font-size serve para definir o tamanho da font e dos elementos utilizados no texto ou palavra no qual queremos mostrar ou destacar.

EXEMPLO:

font-size: 15px

font-size: larger

font-size: 150%


Font Style

=>A propriedade font-style irá definir o estilo da fonte na qual o texto se apresentá, as font-style: mais usadas são italic e normal.

EXEMPLO:

font-style: italic

font-style: normal


Font Weight

=>Está pripriedade tem por objetivo destacar o texto deixanto ele em negrito, ela é mais usada como bolder ou normal. O termo "weith", pode ser traduzido como peso da fonte, ou seja bolder para mais pesado com negrito mais forte e o normal para um negrito mais leve e menos escuro.

EXEMPLO:

font-weight: bolder

font-weight: normal


Font Variant

A propriedade font-variant vai definir o comportamento das letras no texto ou palavra, alternando elas entre tanhos normais ou com caps.Os valores mais usados são: normal e small-caps

EXEMPLO:

font-variant: normal

font-variant: small-caps

Position :

position: static;
position: relative;
position: absolute;
position: sticky;
position: fixed;

O position define uma posição para um elemento dentro do documento.

Static:
Este elemento é o valor padrão, Não possui efeito.

relative:
relative pode reposicionar o bloco deixando o local padrão, não alterando o fluxo normal do documento, como no exemplo a cima.

absolute:
O absolute é parecido com o relative só que, ele muda o fluxo normal do documento, como no exemplo acima ele esta no meio dos outros blocos, mas seu local de origin não é visível, como o relative.

sticky / fixed:
Estas duas propriedades têm o mesmo efeito do relative(sticky) e absolute(fixed), so que eles ficam no visor, exemplo quando você rola a página eles continua na sua tela, mesmo saindo do seu local de origin dos elementos.O sticky troca entre eles sua posição. Aqui possui um exemplo o que estou falando.
obs:(o fixed esta definido com static até você ativalo no botão) ⇒ Referencias: Developer Mozila

Display :

Display é a propriedade do css para definir o formato de renderização de um elemento, o valor padrão é definido muitas vezes pelo navegador mas o no XML é inline.
obs:(todos os exemplos a baixo estão em display inline-block.)

Display: inline;
Display: inline-block;
Display: none;
Display: block;

Inline:
Gera uma ou mais Caixas de elementos em linha.

block:
Gera um caixa de elemento em bloco

none:
Desativa a exibição dos elementos, todos os filhos também tem sua exibição desativada, O documento é renderizado como se não existisse o elemento.

inline-block
O inline-block gera uma caixa de elemento de bloco que fluirá com o conteudo. ⇒ Referencias: Developer Mozila
